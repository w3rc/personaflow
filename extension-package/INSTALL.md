# PersonaFlow Chrome Extension - Installation Guide

## Quick Start

1. **Extract this zip file** to a folder on your computer
2. **Open Chrome** and go to chrome://extensions/
3. **Enable Developer Mode** (toggle in top-right)
4. **Click "Load unpacked"** and select the extracted folder
5. **Sign in** to PersonaFlow at https://crystalknows-clone.vercel.app
6. **Start analyzing** LinkedIn profiles!

## Step-by-Step Instructions

### 1. Install the Extension
- Download and extract the personaflow-extension.zip file
- Open Google Chrome
- Type chrome://extensions/ in the address bar
- Turn on "Developer mode" (toggle switch in top-right)
- Click "Load unpacked" button
- Select the folder where you extracted the extension files
- The PersonaFlow extension should now appear in your extensions list

### 2. Pin the Extension
- Click the puzzle piece icon (Extensions) in Chrome toolbar
- Find "PersonaFlow - LinkedIn Personality Insights"
- Click the pin icon to keep it visible in your toolbar

### 3. Sign In to PersonaFlow
- Go to https://crystalknows-clone.vercel.app
- Create an account or sign in to your existing account
- Make sure you're signed in before using the extension

### 4. Use the Extension
- Navigate to any LinkedIn profile page
- Click the PersonaFlow extension icon in your toolbar
- The extension will show your authentication status
- Click "Extract Profile Data" to analyze the profile
- View results in your PersonaFlow dashboard

## Troubleshooting

**Extension not working?**
- Make sure you're signed in to PersonaFlow first
- Refresh the LinkedIn page and try again
- Check that Developer Mode is enabled in chrome://extensions/

**Can't see extracted profiles?**
- Check your PersonaFlow dashboard at https://crystalknows-clone.vercel.app/dashboard/profiles
- Profiles with "Extension" badges are from the Chrome extension

**Authentication issues?**
- Sign out and sign back in to PersonaFlow
- Make sure you're using the same browser for both the extension and PersonaFlow
